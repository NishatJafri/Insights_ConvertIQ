# ConvertIQ: Smart Conversion Dashboard

## Project Overview  
**ConvertIQ** is a web analytics dashboard built using Power BI to analyze user behavior, traffic sources, and conversion performance.  
The project focuses on identifying key factors that influence conversions and improving overall website performance through data-driven insights.

---

## Objective  
- Analyze **user behavior** and interaction patterns  
- Evaluate performance across different **traffic sources**  
- Track key **conversion metrics** such as conversion rate and funnel drop-offs  
- Identify factors like **bounce rate and engagement** impacting conversions  
- Provide **data-driven insights** for performance optimization  

---

## Dataset Information  
- Dataset sourced from Kaggle  
- Uploaded to GitHub for project reference  

**Key Metrics:**
- Users  
- Sessions  
- Bounce Rate  
- Conversion Rate  
- Traffic Source  
- Engagement Metrics

## Key Features  
- KPI cards showing **Users, Sessions, Conversion Rate, and Bounce Rate**  
- **Conversion funnel** to track user journey and drop-offs  
- **Traffic source analysis** across Organic, Direct, Referral, and Paid channels  
- **User engagement trends** over time  
- Interactive **filters (slicers)** for dynamic analysis  

---

## Key Insights  
- **Organic traffic** shows better conversion performance compared to other channels  
- Lower **bounce rate** is associated with higher conversions and engagement  
- The **conversion funnel** reveals drop-offs at intermediate stages  
- Some channels generate traffic but have lower **conversion efficiency**  
- User activity trends indicate variation in engagement patterns
